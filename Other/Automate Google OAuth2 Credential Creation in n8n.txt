Overview: Automate Your Google OAuth2 Credential Creation

This workflow template streamlines the process of creating and naming individual Google OAuth2  credentials for multiple Google services within n8n. By automating tasks typically done manually, it saves significant time and reduces the risk of errors.

Who Is This For?

This template is perfect for:  
Marketers who frequently use Google Docs, Google Sheets, Google Slides, and Google Drive for document automation.  
Developers or technical users who manage numerous integrations and need to minimize credential creation overhead.  
Teams or solo users with multiple Google accounts who want to eliminate repetitive Google OAuth2 credential creation tasks.

Prerequisite Knowledge

‚Ä¢ Basic familiarity with n8n, including how to configure nodes and connect services.  
‚Ä¢ A Google OAuth JSON file (client ID, client secret, etc.) already set up in your Google Cloud Console.  
‚Ä¢ An understanding of Google services (Docs, Sheets, Slides, Drive, Gmail, Calendar, Contacts and others) and which you may need for your use cases

What Problem Does It Solve?

Manually creating and naming separate OAuth2 credentials for each Google service can be time-consuming and prone to errors. For instance, if you manage 16 Google accounts, you might need to create 112 credentials manually. This workflow automates that process by using your Google OAuth JSON file and email address, generating uniquely named OAuth2 credentials for each Google service in just a single run per account.

Key Features

Manual Trigger ‚Äì Start the workflow on demand to generate new credentials.  
Set Node for Google OAuth Details ‚Äì Store your client ID, client secret, and other JSON info securely in one place.  
Automated Naming Convention ‚Äì Use your Google email address to create distinct names for each credential, simplifying organization.  
Array of Google Services ‚Äì Customize which services to create credentials for, such as Docs, Sheets, Slides, Drive, Gmail, Calendar, or Contacts.  
Batch Credential Creation ‚Äì Split the array of Google services and automatically create separate OAuth2 credentials for each.  
Seamless OAuth2 Integration ‚Äì Effortlessly authenticate all generated credentials within n8n.

How to Set Up

Insert Your JSON Details ‚Äì Paste your Google OAuth 2.0 JSON data into the ‚ÄúGoogle JSON‚Äù node.  
Add Your Google Email Address ‚Äì Specify your email address in the ‚ÄúGoogle Email‚Äù node to personalize the naming convention.  
Configure n8n Account Credentials ‚Äì In the ‚Äún8n Create Credentials‚Äù node, select or add proper credentials (e.g., n8n API credentials).  
Execute the Workflow ‚Äì Run the workflow to automatically generate individual OAuth2 credentials.  
Finalize in n8n ‚Äì Visit your n8n Credentials dashboard to complete the OAuth sign-in for each newly created credential.

Customization Options

Modify the Services Array** ‚Äì Easily add or remove Google products based on your workspace needs by editing the array in the ‚ÄúServices‚Äù node.  
Adjust Naming Logic** ‚Äì Tailor the credential naming convention in the ‚Äún8n Create Credentials‚Äù node to match your organization‚Äôs style or best practices.  
Set Role-Based Access** ‚Äì In n8n, control who can view and use these automatically generated credentials by adjusting user permissions.
